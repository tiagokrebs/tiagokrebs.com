# Github Page Development

Used to build my Github Page with gohugo.io.  
Based on [this post](https://youngkin.github.io/post/createafreeblogsite/).

## Add new post
```
$ hugo new post/newpost.md
```

## Run local server
```
hugo server -D
```
Go to http://localhost:1313/

## Deploy
```
$ ./deploy.sh
```
